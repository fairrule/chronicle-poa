Chronicle Proof of Asset — 수집·분석 대시보드

이게 뭔가요? 현실 자산(미국 국채, 회사 대출 등)을 블록체인 토큰으로 만든 펀드 18개의 검증 데이터를 매일 자동 수집해, ① 지금 얼마인가 ② 데이터를 믿을 만한가 ③ 투자할 만한가 를 보여줍니다. 읽기 전용(거래·송금 없음).
알아둘 용어: NAV=토큰 한 개의 가치(1.03이면 3% 상승) · 오라클=펀드 가치를 블록체인에 올리는 데이터 배달부 · 검증자 2/25=25명 중 2명 서명하면 값 확정.
⚠️ 신뢰 경계 — 이 데이터는 "운용사 보고값이 조작 없이 전달됐다"만 보장합니다. 보고 내용 자체를 독립 회계감사한 것은 아닙니다. 투자 자문이 아닙니다.

① 밸류에이션

자산별 현재 가치(NAV)와 믿을 만한 값인지 보여줍니다. 등급: A=신선·정상, B=조금 오래됨, C=너무 오래되거나 이상. freshness: fresh/stale/expired(갱신 멈춤·주의). 검증자 2/25=25명 중 2명 서명하면 확정. 오른쪽 출처 링크로 블록체인에서 원본 값을 직접 확인할 수 있습니다.

② 리스크 분해

펀드가 담은 실제 증권들의 만기 분포와 가중평균만기(WAM)를 봅니다. 만기가 짧을수록(WAM 작을수록) 금리 변화에 덜 흔들려 안전한 편. look-through(투과) 🎁 = 선물 바구니 두 개를 열었더니 둘 다 사과만 가득이면 사실 몰빵인 것처럼, 여러 토큰의 속을 열어 실제로 어디에 얼마나 노출됐는지 합산합니다(예: 국채에 총 얼마).

③ 무결성 게이트

지금 문제 있는 데이터를 잡아냅니다. 갱신이 멈췄거나(expired), 여러 블록체인에 올라간 같은 자산의 값이 서로 어긋나면(발산) 빨간 경고. 초록은 "여러 체인에서 값이 일치한다 = 교차검증 통과"라는 뜻입니다.

④ 벤치마크

"직접 국채 사는 것보다 나은가?"를 봅니다. 토큰 이자(yield)를 같은 만기 미국 국채 금리와 비교한 차이(스프레드)로 판단: 국채 펀드는 음수(−)=운용·토큰화 비용(직접 국채가 유리). 크레딧·CLO는 양수(+)=신용위험 프리미엄(정상, 위험 감수 대가). 주식은 국채 비교 부적합. 수동 배지=자동이 아니라 사람이 공개 자료에서 넣은 값(가끔 갱신 필요).

실적 (Track Record)

"광고 말고, 실제로 얼마 벌었나"를 봅니다. 과거 가격 이력으로 실현수익·변동성·최대낙폭을 계산. 가장 조심할 지점: 현재 광고 이자와 실제 실현수익이 크게 다를 수 있습니다. 실현이 광고보다 훨씬 낮으면 빨간색(예: "말은 4%인데 실제론 −0.2%"). 변동성·최대낙폭이 낮을수록 안정적입니다.

오라클 신뢰도

"이 데이터 배달부(오라클)를 믿어도 되나"를 0~100점으로 매깁니다. 신선도·검증자 쿼럼·교차체인 일치·가동률을 종합. 점수 옆 플래그가 약점을 숨기지 않고 그대로 보여줍니다(예: "쿼럼 2 약함"). 점수가 높아도 플래그는 꼭 읽어보세요.

출처 (Provenance)

"이 숫자, 어디서 왔나"를 모두 밝힙니다. 온체인 값은 블록체인 탐색기 링크로, 오프체인 값은 원본 API 링크로 직접 대조할 수 있습니다. 믿으라고만 하지 않고 확인할 길을 열어둔 겁니다.
